Overview

Namespace for classes and modules that synchronize and handle customer's billing source.

Billing Source contains information about localization and maximum allowance for test case runs and exploratory tests. This module supports multiple billing source connectors, used to retrieve and synchronize BillingStatement from billing source connected with Organization.
